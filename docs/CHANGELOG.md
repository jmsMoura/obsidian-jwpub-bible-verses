# Changelog

All notable changes to the JW Bible Verses plugin will be documented in this file.

## [1.1.0] - 2025-04-04

### Added
- Support for verse ranges (e.g., "John 1:1-3", "Matthew 5:3-10")
- Verse numbers displayed when inserting verse ranges
- Enhanced URL generation for verse ranges

### Improved
- Better verse extraction for multi-verse content
- More robust error handling for verse range parsing

## [1.0.0] - 2025-04-04

### Added
- Initial release of the JW Bible Verses Plugin
- Insert Bible verses from JW.org
- Auto-convert existing Bible references
- Support for multiple languages
- Customizable formatting for verse display
- Auto-update functionality for verses
- Bible book customization
