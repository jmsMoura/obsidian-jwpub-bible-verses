# JW Bible Verses Plugin for Obsidian

Insert and keep up-to-date Bible verses from JW.org directly in your Obsidian notes.

## Features

- Insert Bible verses from JW.org directly into your notes
- Auto-convert existing Bible references to live linked verses
- Support for verse ranges like "John 1:1-3" or "Salmos 1:1-3"
- Enhanced poetry handling for Psalms/Salmos books
- Distinctive verse number formatting with backticks (`)
- Supports multiple languages (configurable via language codes)
- Automatically detects and handles various Bible reference formats
- Keeps verses up-to-date with the latest content from JW.org

## Installation

### From Obsidian Community Plugins

1. Open Obsidian Settings
2. Go to Community Plugins and click "Browse"
3. Search for "JW Bible Verses"
4. Click Install, then Enable the plugin

### Manual Installation

1. Download the latest release zip file from the [Releases](https://github.com/jmsMoura/obsidian-jwpub-bible-verses/releases) page
2. Extract it to your Obsidian plugins folder: `<vault>/.obsidian/plugins/jwpub-bible-verses/`
3. Enable the "JW Bible Verses" plugin in Obsidian Settings > Community Plugins

## Quick Start

1. Open the command palette (Ctrl+P or Cmd+P)
2. Search for "JW Bible Verses: Insert Bible Verse"
3. Enter a Bible reference like "John 3:16" or "Salmos 23:1-3"

For more detailed instructions, see the [Usage Guide](docs/USAGE.md).
